# AWS Landing Zone Presentation

This folder contains presentation material for explaining the AWS Landing Zone project.

## How To Use

1. Take screenshots from AWS Console, GitHub Actions, and local terminal.
2. Save them in:

```text
presentation/assets
```

3. Use the exact filenames listed in `image-checklist.md`.
4. Follow `storyboard.md` slide-by-slide when building or presenting the PPT.
5. Use `flowchart.mmd` if you want to add an architecture flowchart.

## Current Output

This is a PPT-ready content package. It includes:

- slide titles
- speaker notes
- screenshot placement guidance
- architecture flowchart
- image checklist

No AWS credentials or secrets should be placed in this folder.
